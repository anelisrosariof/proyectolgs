# @luxury-grand-stage/frontend

Interfaz de usuario del sistema Luxury Grand Stage, construida con **Next.js** y **React**.

Provee las pantallas para la gestión de reservas, venta de entradas, facturación, dashboard administrativo y demás módulos del sistema.

## Stack

- Next.js
- React
- Tailwind CSS
- TypeScript

## Desarrollo

```bash
# Desde la raíz del monorepo
pnpm dev:frontend

# O desde este directorio
pnpm dev
```

El frontend se ejecuta en `http://localhost:3000`.
