# @luxury-grand-stage/core

Paquete compartido que contiene schemas, modelos, tipos y utilidades usadas tanto por el frontend como por el backend.

## Uso

Este paquete se consume como dependencia de workspace:

```json
{
  "dependencies": {
    "@luxury-grand-stage/core": "workspace:*"
  }
}
```

```ts
import { /* ... */ } from '@luxury-grand-stage/core';
```
